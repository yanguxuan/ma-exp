## PPT转Word文档任务

将6个多智能体与机器人课程PPT转换为Word知识点整理文档。

### 处理文件
- `1_2-大模型Agent.pptx` → 第1章
- `3_4_5-多智能体.pptx` → 第2章
- `6_7_8-强化学习部分.pptx` → 第3章
- `9-机器人动力学.pptx` → 第4章
- `10-智能机器人.pptx` → 第5章
- `11-实践案例.pptx` → 第6章

### 输出文件
`D:\code\ma-exp\多智能体与智能机器人知识点整理.docx`
- 6章，164个知识点小节
- 约7.4万字符
- 92.7 KB
- A4纸，紧凑边距（适合打印）

### 技术流程
1. 用 `python -m markitdown` 提取各PPT文本到 `D:\Temp\ppt[1-6].txt`
2. 用 `node generate_word.js`（需在项目目录运行以使用本地docx包）生成docx
3. 过滤：去掉图片引用、老师备注Notes、目录/标题页、重复幻灯片、来源标注
4. 同主题幻灯片合并；每章新页开始

### 注意事项
- `/tmp` 在Bash中对应 `D:\Temp`（Windows路径）
- `python` 在此环境指 Python 3.14，路径为 `C:\Python314\python`
- docx模块需在项目目录安装（`npm install docx`），全局安装因ES模块冲突无法跨目录require
